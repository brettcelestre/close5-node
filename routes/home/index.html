Close5 Code Challenge: API Guide

/
- API Guide
/api/recent/
- The entire list sorted by creation date (Ascending and Descending)
/api/price/lowest
- The entire listed sorted by the item’s price Ascending
/api/price/highest
- The entire listed sorted by the item’s price Descending
/api/id/ID_NUMBER
- Any single item by its id number (ID_NUMBER)
/api/userid/ID_NUMBER
- An array of items based on any users id (ID_NUMBER)
/api/range/LAT/LON
- A route to request for items within 50 miles of their location coordinates. LAT = your latitude, LON = your longitude
